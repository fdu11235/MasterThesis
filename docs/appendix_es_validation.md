# Appendix: Validation of Ground-Truth Expected Shortfall
\label{appendix:es-validation}

closed-form ES formulas used as ground truth and two
independent benchmarks: high-precision Monte Carlo (N = 10^7)
and `scipy.integrate.quad` adaptive quadrature on the population
density. Generated by `scripts/validate_es_closedform.py`.
